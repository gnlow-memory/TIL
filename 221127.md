```prolog
a(1, Y).
% ?- a(X, 2).
% X = 1, Y = 1

% unify(term1, term2) -> mgu(Most General Unifier)

% > Find mgu for these two unifications:
% > a(X, Y) | a(b(Y), c(Z))
% 	X = b(c(_Z))
% 	Y = c(_Z)
% > a([1|X]) | a(X)
%	X = [1..]  % Prolog에선 유한해야함
```

```
Query: Rule
Rule: (name, val)
Red: ((name, val), (oname, val))
Cause: Rule | Red
Program: Cause[]

run = program => query => result
  program: Program
  query: Query
  match = program & ($.name = query.value)
  result = run match.
    Rule => (name, val) end!
    Red => (oname, val)
```
```
Rule: (head, body)
Clause:
  | (lhs, rhs)
  | (lhs)
  lhs, rhs: Rule
solve = program => goal => return
  program: Clause[]
  goal: Rule
  match = program..($.head = goal.head)
  return = match.
    []          => False
    (rhs: Val)  => True
    _           => solve match.rhs
```
